# easy-docker-setup
Simple bash scripts to install docker from a single command. *Assuming wget is installed*

## Debian / Ubuntu

To quickly 

```
wget -O - https://raw.githubusercontent.com/OverengineeredOne/easy-docker-setup/main/debian.sh | bash
```
